# Static Assets

This directory contains static assets for the Sports Booking Portal.

## Images

### Sports Images
- `pickleball.jpg` - Pickleball court action
- `volleyball.jpg` - Beach volleyball game
- `football.jpg` - American football field
- `cricket.jpg` - Cricket match
- `tennis.jpg` - Tennis court

All images are optimized for web display with dimensions of 400x200 pixels.

## Usage

Images are referenced in the HTML files using relative paths:
```html
<img src="static/images/pickleball.jpg" alt="Pickleball" class="sport-img">
```
