# Sports Booking Portal

A modern, responsive sports booking website with glassmorphism design and smooth animations.

## Features

- **Login System**: Multi-role login (Player, Coach, Admin)
- **Sports Dashboard**: 5 sports available for booking
  - 🏓 Pickleball
  - 🏐 Volleyball  
  - 🏈 Football
  - 🏏 Cricket
  - 🎾 Tennis
- **Booking System**: Time slot selection with availability checking
- **Glassmorphism Design**: Modern glass-like UI with blur effects
- **Animations**: Smooth transitions and floating shapes
- **Responsive Design**: Works on all devices
- **Local Storage**: Persistent booking data

## Technologies Used

- HTML5
- CSS3 with Glassmorphism effects
- JavaScript (ES6+)
- Bootstrap 5
- Local Storage API

## How to Use

1. Open `index.html` in a web browser
2. Login with any credentials (demo mode)
3. Select a sport from the dashboard
4. Choose date and time slot
5. Confirm booking

## Features Details

### Login Page
- Glassmorphic login form
- User type selection
- Shimmer animation effect
- Demo mode (any credentials work)

### Dashboard
- Responsive grid layout
- Interactive sport cards with hover effects
- User information display
- Logout functionality

### Booking System
- Date picker (minimum date = today)
- Time slot selection (6 AM - 10 PM)
- Duration selection (1-3 hours)
- Player count input
- Real-time availability checking
- Booking confirmation

- Floating background shapes
- Card hover effects
- Smooth transitions

- Pulse animations
